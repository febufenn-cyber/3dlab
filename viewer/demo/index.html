SceneForge embed demo

This is the whole integration — one script, one tag:

<script src="../dist/rf.js"></script>
<rf-walkthrough
    src="./sample/scene.ksplat"
    poster="./sample/poster.png"
    mode="walk"
    height="480px"></rf-walkthrough>

Serve this folder over HTTP (python -m http.server from viewer/) and drop a real scene's scene.ksplat + poster.png into viewer/demo/sample/ — the output of rf-scene build video.mp4 -o out/ works as-is. Until then the tile shows an honest load error instead of a fake scene: SceneForge never invents geometry, including here.
