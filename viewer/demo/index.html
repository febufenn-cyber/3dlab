SceneForge embed demo

This is the whole integration — one script, one tag:

<script src="../dist/rf.js"></script>
<rf-walkthrough
    src="./sample/scene.ksplat"
    poster="./sample/poster.png"
    mode="walk"
    camera-position="0.7,0.6,1.5"
    look-at="2.5,1.8,1.0"
    height="480px"></rf-walkthrough>

The bundled sample is a procedurally generated synthetic room (viewer/tools/make-sample-scene.py) — not a real capture, and labeled as such on the poster too: SceneForge never passes generated content off as a reconstruction. A real captured scene lands here after the first GPU run (see QUALITY.md); the output of rf-scene build video.mp4 -o out/ drops into viewer/demo/sample/ as-is. Serve this folder over HTTP (python -m http.server from viewer/).
