Pipeline

One video in. Three artifacts out.

Upload goes straight to storage over a presigned URL — the control plane only ever moves JSON. A GPU worker picks the job up and reports honest state back — including failure, with the exact reason.

🎥

Capture & check

Frames are extracted, blur- and coverage-gated. A capture that can't reconstruct is rejected up front with a reason code and re-shoot guidance — not silently "fixed."

📐

Feed-forward geometry

lingbot-map (Apache-2.0) is built to recover camera poses and world points in one forward pass — seconds-scale by design, with measured timings due from the first GPU acceptance run. Classical COLMAP remains the verified fallback; a confidence gate keeps failures honest.

✨

Walkable splat

gsplat MCMC training under a hard size budget → a compact .ksplat that streams progressively, sized for mid-range phones (frame-rate acceptance is part of the pending hardware run).

🧠

Scene understanding

Rooms, walls, door and window openings, object boxes — open-vocabulary detection plus occupancy-grid geometry, in one frozen JSON contract.

📏

Metric scale, stated honestly

Scale comes from door priors, your known dimension, or camera height — each with a confidence and a tolerance the UI must show. No fake precision.

🗺️

Auto floor plan

An SVG plan with room areas and a scale bar, drawn from the semantic graph — ready for listings, handovers, and quotes.
